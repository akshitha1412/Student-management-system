# Student Management System

## 📌 Description
A simple CRUD-based Student Management System built using Spring Boot.

## 🚀 Features
- Add Student
- View Students
- Delete Student

## 🛠 Tech Stack
- Java
- Spring Boot
- MySQL

## ▶️ Run Project
1. Clone repo
2. Configure DB in application.properties
3. Run main class

## 📡 API Endpoints
GET /students  
POST /students  
DELETE /students/{id}
